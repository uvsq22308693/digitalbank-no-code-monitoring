# DigitalBank – Plateforme No-Code / Low-Code de Monitoring

## Description
Projet de groupe – Partie 2  
Plateforme de gestion et monitoring de données bancaires pour DigitalBank.

## Objectifs
- Visualisation des données clients et transactions
- Détection de fraude en temps réel
- Gestion des accès (RBAC)
- Monitoring sécurité et performances
- Automatisation des alertes

## Technologies prévues
- PostgreSQL
- Supabase / Hasura
- Metabase / Retool / Grafana
- n8n / Make.com
- Flask (API ML)
- Docker
